## ⚙️ How It Works

### Signup Process

1. User enters:
   - Username
   - Email
   - Password

2. Form validation checks:
   - Empty fields
   - Valid email format
   - Password length

3. If all validations pass:
   - User details are stored temporarily
   - Success message is displayed
   - Form fields are cleared

### Login Process

1. User enters:
   - Username or Email
   - Password

2. System checks:
   - Whether an account exists
   - Username/Email correctness
   - Password correctness

3. If credentials are correct:
   - Login successful message is displayed
   - Form fields are cleared

4. If credentials are incorrect:
   - Appropriate error messages are shown
